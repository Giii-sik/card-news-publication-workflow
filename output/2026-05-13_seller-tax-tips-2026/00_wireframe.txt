## 현재 결정

- Hook / Intro는 현재 방향 유지.
- Info는 Card 03을 대표 카드로 먼저 다듬고, 같은 규칙을 04~09에 순차 적용.
- Card 03~09는 텍스트형 + 상단 Fluent flat 스타일 2D 아이콘으로 구성.
- CTA는 템플릿 공유 시 제거 (브랜드 마무리 카드는 사용자 자기 자산으로 추가).

## 공통 레이아웃 가이드

- Canvas: 1080 × 1350
- Safe area: info text 기준 좌우 100px, 상하 90px
- Info image slot: 사용 안 함
- Info text area: x 100~980, padding 100px, 중앙 정렬
- Info headline: 62px / 1.35 / 700
- Info body: body2 token / 최대 4줄
- Footnote: 내용 출처는 카드 내부 미표기

## 줄바꿈 원칙

- 어색한 축약 줄바꿈 금지.
- 한 줄은 안전영역 안에서 가능한 만큼 채운다.
- 단어/의미가 깨지는 지점보다 문장 호흡이 자연스러운 지점에서 끊는다.
- Info body는 3줄 기준. 더 필요하면 카피를 줄이거나 카드를 분리한다.
