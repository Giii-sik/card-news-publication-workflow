# Learnings — 누적 회고 (템플릿)

> 두 도구(Claude Code / Codex)가 공유하는 학습 누적 파일.
> **매 세션 시작 시 이 파일을 먼저 읽고**, 세션 종료 직전 새 패턴을 추가한다.

분류 라벨 예시:
- `[research]` 토픽 리서치 패턴 (효과적/폐기 쿼리, 토픽 각도)
- `[wireframe]` 와이어프레임·줄바꿈·길이 패턴
- `[image]` 이미지 컨셉·프롬프트·트랙 선택
- `[tone]` 카피 톤·금지어
- `[workflow]` 운영 패턴 (핸드오프, 게이트, 컨펌)
