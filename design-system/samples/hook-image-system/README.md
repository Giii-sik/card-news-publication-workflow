# Hook Image System Sample

This folder is the working sample for the Brand hook image generation system.
Use it to compare and refine the main hook image styles while keeping
typography, logo, gradient, and final composition in HTML/CSS.

## Status

- Confirmed: hook typography system.
- Confirmed: 3D/diorama hook image style.
- Confirmed: pixel illustration hook image style.
- Removed from hook: reality-composite style. Use
  `design-system/samples/info-image-system/` for photo/reality-composite info images.

## Outputs

- `output/01_hook_3d.png` — 3D/diorama hook.
- `output/01_hook_illustration.png` — pixel illustration hook.

## Source Assets

- `assets/01-hook-bg-3d-clean.png`
- `assets/01-hook-bg-illustration-clean.png`

These are clean source images. Final gradient, logo, and Korean headline are
composed in HTML/CSS.

## Purpose

This is not a one-off carousel sample. It is a visual system sample for future
hook images. Keep clean source images, HTML previews, and rendered PNGs here.
Reusable prompt records and production recipes live in
`design-system/image-production/hook/`, and reusable style rules live in
`design-system/image-production/styles/`.
