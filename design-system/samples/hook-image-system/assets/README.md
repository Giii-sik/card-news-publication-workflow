# Hook Image System Assets

Clean generated image assets for confirmed Brand hook image styles.

Current hook styles are limited to 3D/diorama and illustration. Reality or
photo-composite images belong in `design-system/samples/info-image-system/`.

Rules:

- no Korean text
- no baked Brand logo
- no baked bottom gradient
- no final card layout

Use these as style references for future hook image generation. Final
typography, logo, gradients, and overlays belong in HTML/CSS.

## Prompt Records

Prompt records are managed centrally in
`design-system/image-production/hook/`. This assets folder should stay focused
on clean source images used by the sample.
