# Samples (`design-system/samples/`)

Samples are filled examples — 카드 레이아웃·hook 이미지·info 이미지의
시각 기준이 되는 채워진 PNG와 HTML.

Use this folder to judge whether a layout, image style, typography scale, or
carousel flow feels right after real or dummy content has been placed inside.

## Rule

- Real/dummy text가 들어가고 렌더된 PNG가 있으면 `design-system/samples/`에 둠.
- 명칭은 samples 폴더와 design-system 본문 .md(`hook-image-system.md` /
  `info-image-system.md`) 자매 파일이 1:1로 매칭됨 — 시각 기준은 여기,
  스타일 룰은 본문 .md에서.

## Current Samples

- `card-layout-system/` — dummy-content preview of the core card layout
  families.
- `hook-image-system/` — confirmed hook image style system for 3D/diorama and
  illustration source images.
- `info-image-system/` — image samples for info-card visual slots, including
  reality-composite/photo-composite directions.
