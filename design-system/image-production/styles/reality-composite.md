# Reality Composite

Style and production rules for making Brand reality-composite images.

## Style Definition

Use for info-card images that feel like an editorial photo composite:
real-world texture, believable lighting, and one clear explanatory metaphor.

This style is no longer a hook style. It should support the card text from the
info image area, not carry the full hook message.

## Best For

- brand case studies
- seller pain or market tension
- real product/place references
- explanatory metaphors
- before/after visual ideas
- policy, fee, or settlement changes that need urgency

## Core Production Pattern

- Use photorealistic people, objects, and lighting.
- Compose the image like an editorial/news thumbnail, but keep it clean enough
  to sit above the Brand text box.
- Use one clear metaphor that can be understood in one second.
- Recent reference direction: a person anchors the emotion, while one
  exaggerated prop or situation explains the topic before the text is read.
- Keep the scene believable even when the metaphor is exaggerated.
- Use practical ecommerce objects: receipts, calculator, laptop, parcel box,
  shopping bag, settlement documents, charts, storefront-like props.
- Use final typography and card layout in HTML/CSS, not inside the source
  image.

## Composition Pattern

- Foreground: seller, founder, operator, or concrete ecommerce object.
- Background: exaggerated but believable metaphor for the problem.
- Lighting: cinematic, moody, high contrast, with enough dark space for logo and
  headline overlays.
- Camera: medium close-up or staged editorial scene, cropped for a `1080 x
  819` info image area.

## Prompt Blocks To Reuse

- `photorealistic editorial composite`
- `cinematic ecommerce office`
- `premium news-thumbnail energy`
- `one clear seller/business metaphor`
- `foreground sharp, background slightly softer`
- `clean enough for a lower text box underneath`

## Common Risks

- Looking too dramatic or threatening for a seller education post.
- Using famous-person likenesses or recognizable public figures.
- Adding readable text to documents, signs, or screens.
- Making the image repeat what the body text already says.
- Losing the main subject after the top `1080 x 819` crop.
