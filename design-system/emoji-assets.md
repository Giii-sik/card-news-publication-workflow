# Emoji Asset Rule

Use this note when choosing small emoji-like support graphics for text-only
info cards.

## Default Rule

**Always use Tossface for emoji slots.**

- Use Tossface webfont for quick samples and production emoji slots.
- Use the original Tossface output only. Do not edit, trace, recolor, or turn
  Tossface artwork itself into a derivative asset.
- Good for: clean Toss-like 2D emoji, simple business/card-news accents.

## Feed Credit Rule

Every published post that uses Tossface must include Tossface credit in the
feed caption. Do not put this credit inside the card unless the user explicitly
asks for card-level source credit.

Default feed credit copy:

```text
이 게시물에는 토스팀에서 제공한 토스페이스가 적용되어 있습니다.
```

## 3D Support Object

이모지 cue 위에 더 임팩트 있는 3D 시각이 필요하면 별도 트랙으로 처리한다 —
이모지 객체를 직접 만들지 않고 일반 `3d-graphic` 트랙을 사용한다. 자세한 룰은
[`design-system/image-production/styles/3d-graphic.md`](image-production/styles/3d-graphic.md).
